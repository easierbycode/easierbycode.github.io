Product Variant Data Capture

  • The tool should capture all product variables (color, size, toe type, gender, pack quantities, etc.) for items with multiple SKUs ‣
  • Example discussed: work boots with steel-toe/non-steel-toe, brown/black colors, and men's/women's sizes ‣
  • Some items may have up to ~50 variables; engine should capture everything upfront so retroactive matching is possible ‣
  • Weighted hula hoop flagged as an edge case: a single listing bundles drastically different products (hula hoop + battle ropes) at different prices with different photos — the tool needs to handle pulling the correct variant ‣ ‣ ‣

TikTok Shop Order History Data

  • Participants compared TikTok Shop data exports; one person's export includes order history with retail prices and shipping, another's did not include sample orders ‣ ‣
  • Samples may appear as $0 in order history; if nothing shows as $0, samples either aren't included or aren't labeled as such ‣
  • The export is a .txt file (part of a 10–12 file zip) accessible via phone, extractable to Google Drive ‣ ‣
  • Using an API service to pull data directly into the database was floated as a cleaner alternative ‣

Affiliate Shareable Image Feature

  • Flow: user scans QR code on the website → authorizes via TikTok → tool pulls top free product values and generates a shareable image ‣ ‣
  • Image displays the dollar value of free products received, grouped by category (e.g., apparel, home goods, snacks) rather than listing specific items ‣ ‣
  • "Got" vs. "received" wording debated — tied 2–2, tiebreaker needed ‣
  • Option discussed to show the agency's cumulative total given away (potentially millions) to add social proof ‣
  • Displaying specific product names/items deemed optional — should only show if value is meaningful (e.g., threshold of $100+); otherwise it may underwhelm (e.g., "a bag of mushroom gummies") ‣ ‣ ‣
  • Less information on the image is preferred to drive curiosity and encourage people to reach out to the affiliate ‣

Influencer Agency Concept & Marketing

  • The "influencer agency" term positions regular people as gaining access to free products through the agency — a loophole normally only available to influencers ‣ ‣
  • The concept leverages the widely understood idea that influencers receive free product, making the pitch intuitive ‣ ‣
  • Affiliates are required to create more content than TikTok's baseline (5 videos vs. 1), which builds repeat brand relationships and avoids burning through them ‣
  • CTA on the shareable image: "DM me your email" preferred over a generic call-to-action to avoid comment-baiting penalties on Facebook ‣ ‣ ‣
  • On response, affiliates would receive a link to a video sales page and their custom affiliate link ‣ ‣

Social Sharing & Branding

  • Social sharing plugins exist for all major platforms; clicking share will pre-populate the image and require only a login from the user ‣
  • Instagram story format (vertical) suggested as a good fit — content refreshes naturally every time new samples are received ‣ ‣
  • Allowing multiple template backgrounds with swappable text was suggested to give affiliates customization ‣
  • Lifepreneur branding on the image debated: branding helps the agency but provides no perceived value to the affiliate or their audience; leaning toward no branding for now ‣ ‣
  • Adding a profile picture to the image would make it feel personal and not like a company advertisement ‣
