PROJECT SUMMARY
This is NOT a Phaser game despite being in a directory with Phaser plugin files. It's a high-performance vanilla JavaScript Canvas platformer featuring:
Custom physics engine
Frame-based animation system
Parallax scrolling background with sprite atlas
Full input handling (keyboard, gamepad, touch, mouse)
Mobile-optimized with haptic feedback
Responsive level loading from JSON
Asset loading with fallback rendering
